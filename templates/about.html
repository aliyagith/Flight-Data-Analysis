About Airplane Crash Analytics

This project aims to analyze airplane crashes and fatalities since 1908, providing valuable insights into aviation safety trends over more than a century.

By visualizing crash locations, analyzing aircraft types involved, and identifying temporal and geographical patterns, this project seeks to contribute to a better understanding of factors affecting aviation safety.

Data Source

The dataset used in this project is sourced from [Your Data Source Here]. It includes detailed records of airplane crashes, including dates, locations, aircraft types, operators, and fatalities.

Purpose

The primary purpose of this project is to:

  • Provide a historical overview of airplane crashes.
  • Identify trends and patterns in aviation safety.
  • Offer interactive visualizations to explore the data.
  • Support aviation safety improvements through data-driven insights.

Technologies Used

  • Flask: Web framework for Python.
  • Tailwind CSS: Utility-first CSS framework for styling.
  • Plotly: Interactive graphing library.
  • Pandas: Data manipulation and analysis.
